APISCAN v1.0 – AI-enhanced OWASP API Security Scanner

Extensible scanner for the OWASP API Security Top 10 (2023). Supports Swagger/OpenAPI, multiple authentication flows (incl. OAuth2 + Bearer + mTLS + Api-key and more), realistic dummy requests, parallel scanning, and an optional AI review mode.

⭐ Star on GitHub What's new Version 1.0 Release · MIT

What's new in v1.0

Expanded Auth

--flow now supports none, token, client, basic, ntlm, auth with --client-id/--client-secret, --token-url, --auth-url, --redirect-uri, --scope and optional mTLS (--client-cert/--client-key, --cert-password). Add --insecure for lab use.

Faster Scans

Thread‑pool parallelism via --threads (cap 20) and HTTP connection pooling for higher throughput.

AI‑Assisted Audit

API11 optional analysis: --api11 runs an AI pass over discovered endpoints and saves AI-api11_scanresults.json.

Smart Swagger

Safer dummy mode (--dummy) and a variables template exporter --export_vars PATH (YAML/JSON) driven by OpenAPI schemas.

Rich Reporting

Per‑risk HTML reports + an auto‑generated combined_report.html, log folder per run, and a terminal risk summary.

Connectivity Guard

Pre‑flight reachability checks with retries and clear error exits for auth and TLS issues.

Key Features

OWASP API Top 10

Covers API1..API10 with dedicated modules, plus AI‑assisted review (API11).

Swagger/OpenAPI‑aware

Parses OpenAPI specs, generates realistic dummy payloads, and maps operations automatically.

Multi‑Auth & mTLS

Token, OAuth2 client credentials & auth code, Basic, NTLM, API key header, and client‑cert auth.

Reports

Per‑risk HTML + unified combined_report.html with deduped highlights.

Variables Export

Export a fill‑in template of all parameters & body fields from your OpenAPI.

python apiscan.py --url https://api.example.com \
--swagger openapi.json \
--export_vars vars_template.yml

Dummy Mode

Schema‑aware, consistent values for safe testing without touching real data.

python apiscan.py --url https://api.example.com \
--swagger openapi.json --dummy

Quick Start

# 1) Install (Python 3.11+ recommended)
pip install -r requirements.txt

# 2) Run with a Swagger/OpenAPI file
python apiscan.py --url https://api.example.com \
  --swagger openapi.json --flow token --token <BEARER>

# Export variables template (YAML/JSON)
python apiscan.py --url https://api.example.com \
  --swagger openapi.json --export_vars vars_template.yml

CLI Flags (v1.0)

--url                 Base API URL
--swagger             Path to OpenAPI/Swagger JSON
--flow                none | token | client | basic | ntlm | auth
--token               Bearer token (use with --flow token)
--basic-auth          user:pass
--apikey              API key value
--apikey-header       Header for API key (default: X-API-Key)
--ntlm                DOMAIN\user:pass
--client-cert         PEM client certificate (mTLS)
--client-key          PEM private key (mTLS)
--cert-password       Private key password (optional)
--insecure            Disable TLS verification (DANGEROUS)
--client-id           OAuth2 Client ID
--client-secret       OAuth2 Client Secret
--token-url           OAuth2 Token endpoint
--auth-url            OAuth2 Authorization endpoint
--redirect-uri        OAuth2 redirect URI
--scope               OAuth2 scope(s), space-separated
--threads             Parallelism (default 2, max 20)
--debug               Verbose logging
--dummy               Enable dummy payloads
--export_vars PATH    Write a variables template and exit
--api1..--api11       Run specific audit modules
